# Beyond Accuracy: Multi-Metric Evaluation of Grad-CAM Interpretability for Binary EuroSAT Classification

This repository provides the paper and related resources for the following study:

**Beyond Accuracy: Multi-Metric Evaluation of Grad-CAM Interpretability for Binary EuroSAT Classification**  
Jiyeon Kim  
Journal of Artificial Intelligence Research and Applications, Vol. 2, No. 1, 2025, pp. 91–103.

---

## Paper
- Official publication:  
  https://www.e-jara.org/journal/view.html?pn=current_issue&uid=1534&vmd=Full

---

## Overview
This study investigates the interpretability of Grad-CAM in binary satellite image classification using the EuroSAT dataset.  
Rather than relying solely on classification accuracy, the research evaluates explanation quality through multiple quantitative metrics, providing a more comprehensive assessment of model interpretability.

---

## Key Contributions
- Proposes a multi-metric evaluation framework for Grad-CAM
- Applies interpretability evaluation to binary EuroSAT classification
- Demonstrates limitations of accuracy-based evaluation
- Provides insights into explanation faithfulness

- EuroSAT (binary classification setting)
